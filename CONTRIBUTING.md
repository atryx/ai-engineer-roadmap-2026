# Contributing to AI Engineer Roadmap 2026

Thanks for helping improve this roadmap!

## How to Contribute

1. **Fork** the repo
2. **Create a branch** (`git checkout -b improve/section-name`)
3. **Edit** the README — add resources, fix outdated info, or suggest new sections
4. **Submit a PR** with a clear description

## Guidelines

- Focus on **practical AI engineering**, not academic ML theory
- Prioritize tools and patterns used in **production** AI apps
- Link to **free resources first**, paid ones second
- No affiliate links

## What We're Looking For

- Updated model and tool recommendations
- Real-world project ideas
- New patterns (agents, multi-modal, etc.)
- Better learning resource links
